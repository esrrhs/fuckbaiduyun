# fuckbaiduyun
百度网盘快速加解密工具，防止文件被和谐

## 特性
1. go+qt实现跨平台，目前有mac、windows版本
2. rc4按块加密文件
3. 自动拆分文件，适应百度网盘限制
4. 单线程执行，不耗尽系统资源
5. 进度条显示
6. 保存上次选项
7. 只对输出目录写操作，保证输入目录安全

## 用法
1. 打开程序
2. 选择输入路径
3. 选择输出路径
4. 输入密码
5. 选择切分文件大小
6. 点击GO运行
