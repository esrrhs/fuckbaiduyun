# fuckbaiduyun
百度网盘快速加解密工具，防止文件被和谐

## 用法
拷贝到目录下，直接运行，自动加解密所有文件，加密完手动上传百度云，然后下次从百度云下载下来再运行下解密

## 下载
https://github.com/esrrhs/fuckbaiduyun/releases
